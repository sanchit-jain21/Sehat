# Sehat
 A Multilingual Telemedicine & Digital Health Platform

• Next.js 14, React, TypeScript, Tailwind CSS, PWA, WebRTC, WebSockets, Firebase, FastAPI, Python, Scikit-learn,
XGBoost, PostgreSQL, Docker, GCP

• Built a scalable full-stack telemedicine platform using Next.js 14, React, and Tailwind CSS with PWA support
and multilingual UI (i18next), enabling remote healthcare access and offline health records.

• Implemented low-latency video consultations using WebRTC and WebSockets, and integrated Google Maps
APIs for ambulance requests and real-time location tracking.

• Developed an AI-powered symptom checker using Python, Pandas, NumPy, Scikit-learn, XGBoost, Random
Forest, and Logistic Regression, performing EDA, feature engineering, and model evaluation.

• Built secure backend services with FastAPI and Firebase, enabling real-time pharmacy stock updates, REST
APIs, and JWT-based authentication with PostgreSQL.
